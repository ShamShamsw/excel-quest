# 🌱 Beginner Path — Start Your Excel Quest

**Level:** Starter | **Challenges:** 3 | **Total Time:** ~75–105 minutes

This path is designed for **complete beginners** — no prior Excel experience required. Each challenge builds on the last, introducing one new concept at a time.

---

## Your Journey

### 🥇 Step 1 — [Duplicate Detective](../challenges/data-cleaning/duplicate-detective/)
**Difficulty:** 🟢 Apprentice | **Time:** 20–30 min

Start with data cleaning — before you can analyze data, you need to trust it. This challenge teaches you how to spot and remove duplicates, a foundational skill every Excel user needs.

**Skills:** `COUNTIF`, conditional formatting, Remove Duplicates

---

### 🥈 Step 2 — [VLOOKUP Treasure Hunt](../challenges/formulas/vlookup-treasure-hunt/)
**Difficulty:** 🟢 Apprentice | **Time:** 20–30 min

With clean data in hand, learn to look things up across tables. VLOOKUP is one of the most-used Excel functions — and after this challenge, you'll know exactly how to use it.

**Skills:** `VLOOKUP`, `IFERROR`, exact match

---

### 🥉 Step 3 — [Pivot Pyramid](../challenges/pivot-tables/pivot-pyramid/)
**Difficulty:** 🔵 Journeyman | **Time:** 30–45 min

Now that you can clean and look up data, learn to summarize it. Pivot Tables are Excel's superpower for turning raw rows into meaningful summaries — by region, product, month, and more.

**Skills:** Pivot Tables, grouping, slicers, calculated fields

---
