# 🤖 Automation Path — Excel on Autopilot

**Level:** Advanced | **Challenges:** 4 | **Total Time:** ~135–180 minutes

This path is designed for users who want to **automate repetitive Excel tasks** using formulas, macros, and Power Query. You'll go from efficient lookups all the way to fully automated multi-source data pipelines.

---

## Your Journey

### Step 1 — [VLOOKUP Treasure Hunt](../challenges/formulas/vlookup-treasure-hunt/)
**Difficulty:** 🟢 Apprentice | **Time:** 20–30 min

Start with the fundamentals: automating data lookups with VLOOKUP so you never have to manually scan a spreadsheet again.

**Skills:** `VLOOKUP`, `IFERROR`, exact match

---

### Step 2 — [INDEX-MATCH Maze](../challenges/formulas/index-match-maze/)
**Difficulty:** 🔵 Journeyman | **Time:** 30–45 min

Level up your lookup automation. INDEX-MATCH handles situations VLOOKUP can't, making your formulas more robust.

**Skills:** `INDEX`, `MATCH`, left-lookup, two-way lookup, dynamic references

---

### Step 3 — [Macro Mission](../challenges/automation/macro-mission/)
**Difficulty:** 🟣 Master | **Time:** 45–60 min

Automate formatting and report generation with VBA Macros. One click, perfect output, every time.

**Skills:** VBA Macros, Macro Recorder, loops, conditional logic in VBA

---

### Step 4 — [Power Query Quest](../challenges/advanced/power-query-quest/)
**Difficulty:** 🟣 Master | **Time:** 45–60 min

The ultimate automation: automate the entire data import and transformation pipeline. Combine messy multi-source data into one clean master table — automatically refreshable.

**Skills:** Power Query, merging queries, unpivoting, data types, transformations

---
