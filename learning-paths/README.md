# 🗺️ Learning Paths

Learning paths are curated sequences of Excel Quest challenges designed to help you build skills progressively — like leveling up in a game.

---

## How to Follow a Path

1. Open the path file below that matches your goal
2. Start with the first challenge in the list
3. Complete each challenge (read the README, attempt it in Excel, then check the `solution.ipynb`)
4. Move to the next challenge in the path when you feel confident
5. Take your time — there's no timer!

## Mix and Match

Paths are **suggestions**, not requirements. You're free to:
- Jump directly to any challenge that interests you
- Revisit earlier challenges to deepen your understanding
- Combine challenges from multiple paths

The only rule is to have fun and keep leveling up! 🎮
