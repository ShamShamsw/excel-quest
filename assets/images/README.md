# 🖼️ Images

This folder is a placeholder for project images, including:

- **Banner image** — A game-themed hero banner for the README and static site
- **Icons** — Category icons or challenge artwork
- **Screenshots** — Screenshots of Excel challenges or the static site

---

## Contributing Images

Have some artwork or screenshots to add? We welcome contributions!

- Preferred formats: `.png`, `.svg`, `.jpg`
- Banner dimensions: 1280×640px recommended
- Keep file sizes reasonable (compress images before committing)

Open a Pull Request and add your images to this folder. See [CONTRIBUTING.md](../../CONTRIBUTING.md) for the full contribution guide.
