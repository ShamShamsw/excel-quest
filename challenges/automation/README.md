# 🤖 Automation

Learn to automate repetitive Excel tasks using Macros and VBA.

---

## Challenges

| Challenge | Difficulty | Description |
|-----------|------------|-------------|
| [Macro Mission](macro-mission/) | 🟣 Master | Record and edit a VBA Macro to auto-format a multi-sheet report like a secret agent |

---

## Skills Covered

- Recording Macros with Excel's Macro Recorder
- Opening and editing the VBA Editor (`Alt+F11`)
- Writing VBA loops to process multiple sheets
- Conditional logic in VBA (`If...Then...Else`)
- Assigning Macros to buttons
