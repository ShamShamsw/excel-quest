# 📊 Pivot Tables

Learn to summarize, group, and slice large datasets using Excel's powerful Pivot Table feature.

---

## Challenges

| Challenge | Difficulty | Description |
|-----------|------------|-------------|
| [Pivot Pyramid](pivot-pyramid/) | 🔵 Journeyman | Summarize sales data across regions, products, and months for the ancient architects |

---

## Skills Covered

- Creating Pivot Tables from raw data
- Dragging fields to Rows, Columns, and Values areas
- Grouping date fields (by month, quarter, year)
- Adding Slicers for interactive filtering
- Calculated fields in Pivot Tables
