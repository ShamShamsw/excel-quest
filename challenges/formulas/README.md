# 📐 Formulas

Master the art of Excel formulas — from basic lookups to advanced dynamic references.

---

## Challenges

| Challenge | Difficulty | Description |
|-----------|------------|-------------|
| [VLOOKUP Treasure Hunt](vlookup-treasure-hunt/) | 🟢 Apprentice | Use VLOOKUP to match chest IDs to their contents and find the buried gold |
| [INDEX-MATCH Maze](index-match-maze/) | 🔵 Journeyman | Escape the data labyrinth using INDEX-MATCH for left-lookups and two-way references |

---

## Skills Covered

- `VLOOKUP` (exact and approximate match)
- `IFERROR` for error handling
- `INDEX` and `MATCH`
- Two-dimensional lookups
- Left-lookups (which VLOOKUP cannot do)
- Dynamic cell references
