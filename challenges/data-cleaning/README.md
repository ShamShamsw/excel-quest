# 🧹 Data Cleaning

Learn to identify, remove, and prevent data quality issues in Excel datasets.

---

## Challenges

| Challenge | Difficulty | Description |
|-----------|------------|-------------|
| [Duplicate Detective](duplicate-detective/) | 🟢 Apprentice | Find and eliminate duplicate case files from the city's crime database |

---

## Skills Covered

- `COUNTIF` for identifying duplicates
- Conditional formatting to highlight duplicates visually
- Excel's built-in **Remove Duplicates** tool
- Data validation to prevent future duplicates
