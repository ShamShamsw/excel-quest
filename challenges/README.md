# 📂 Challenges

Welcome to the Excel Quest challenge library! This folder contains all challenges organized by category.

---

## 📁 Folder Structure

Each challenge follows this structure:

```
challenges/<category>/<challenge-name>/
├── README.md        # Challenge description, story, hints, and success criteria
└── solution.ipynb   # Jupyter Notebook with step-by-step solution walkthrough
```

> **Note on `.xlsx` files:** Binary Excel starter files are not committed to this repository. Each challenge README explains what data is needed. The `solution.ipynb` for each challenge generates sample data programmatically using `pandas`, so you can follow along without a starter file.

---

## 🏷️ Naming Conventions

- Category folders use **kebab-case** (e.g., `data-cleaning`)
- Challenge folders use **descriptive kebab-case names** (e.g., `duplicate-detective`, not `challenge-03`)
- Names are chosen to be evocative and game-flavored
