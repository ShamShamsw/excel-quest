## 💡 Tips

- On **laptops**, you may need to press `Fn` along with function keys (`F1`–`F12`) depending on your keyboard settings.
- Use **`Alt`** in Windows to activate the ribbon key tips — press `Alt`, then the shown letters to access any ribbon command via keyboard.
- Many shortcuts work the same in **Google Sheets** (e.g., `Ctrl + C/V/Z`, `Ctrl + F`, `Ctrl + B/I/U`), so these skills transfer.
